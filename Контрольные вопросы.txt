2. Чем Redis отличается от PostgreSQL?
Redis хранит информацию в оперативной памяти с помощью ключей, что позволяет быстро читать и записывать её. Он использует свои специальные команды. PostgreSQL сохраняет информацию с помощью таблиц, и его скорость чтения и записи намного ниже. Для запросов использует SQL
3. Как в Docker Compose настроить зависимость запуска (сначала БД, потом приложение)?
Использовать атрибут depends_on с условием для БД, что приложение не запустится, пока она не будет готова принимать подключения
